# Guessing Game (Rust)

This project implements the classic guessing game from Chapter 2 of *The Rust Programming Language*.

## How to Run

```bash
git clone https://github.com/YOUR_USERNAME/guessing_game.git
cd guessing_game
cargo run
```

## What It Does

- Generates a random secret number between 1 and 100.
- Gives 5 attempts to guess the number.
- Reports if each guess is too low, too high, or correct.
- Gives a small hint after each wrong guess.
- Reveals the number after 5 unsuccessful attempts.
